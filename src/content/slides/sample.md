# マークダウンスライド

ポートフォリオサイトで使えるスライド機能

`---` で区切るだけでスライドが作れます

---

## 特徴

- **マークダウン**で記述
- キーボード操作（← →）でページ送り
- フルスクリーン対応
- レスポンシブデザイン

---

## コードも書ける

```javascript
const greet = (name) => {
  return `Hello, ${name}!`;
};

console.log(greet("World"));
```

---

## リストの例

1. Next.js App Router
2. Tailwind CSS
3. MDX Remote
4. gray-matter

> マークダウンの機能がそのまま使えます

---

## 画像も使える

通常のマークダウン記法で画像を埋め込めます

```markdown
![Alt text](/path/to/image.png)
```
